# Simulador de Comida Rápida con Elementos Compartidos y Sistema Distribuido

Este proyecto implementa un simulador de un restaurante de comida rápida donde múltiples clientes y empleados interactúan simultáneamente.  
El sistema está dividido en dos partes principales:

- **Servidor distribuido**, que administra la lógica del restaurante.
- **Clientes distribuidos**, que se conectan por red para interactuar con el sistema.

---

## Cómo ejecutar el código

### 1. Ejecutar el servidor
Dentro de la carpeta **distributedServer**, usa los siguientes comandos:

- javac Main.java
- java Main bash 

### 2. Ejecutar los clientes
Dentro de la carpeta **distributedClient**, usa los siguientes comandos:

- javac Main.java
- java Main bash
